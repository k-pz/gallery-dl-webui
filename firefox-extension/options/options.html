gallery-dl-webui — Settings

Backend setup

The backend must allow this extension's origin (CORS). Set one of the following environment variables when starting the backend:

  • WEBUI_CORS_ORIGIN_REGEX=moz-extension://.* (easiest — accepts any Firefox extension install)
  • WEBUI_CORS_ORIGINS=<your extension origin> (strict — uses the origin shown below)
