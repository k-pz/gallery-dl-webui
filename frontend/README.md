# Frontend

React + TypeScript + Vite + Mantine + TanStack Query.

See the [top-level README](../README.md) for the full dev workflow.

## Common commands

```sh
pnpm install        # one-time
pnpm dev            # vite dev server on :5173, proxies /api to :8000
pnpm build          # type-check + production bundle into dist/
pnpm generate       # regenerate src/api/* from the running backend's OpenAPI
pnpm lint           # biome check
pnpm format         # biome format
pnpm typecheck      # tsc --noEmit
```

The `src/api/` directory is generated by `@hey-api/openapi-ts` and should not
be edited by hand. Run `pnpm generate` against a live backend to refresh it
after changing the API.
